# FF Lifemanager

**Persönlicher Lebensplaner — lokal, privat, kein Abo.**

Dein All-in-One Life Manager für macOS und Windows:
- 🎯 Jahres- & Langzeitziele (Big 5 for Life)
- 💰 Finanzen: Vermögen, Fixkosten, Einkommen
- 🌟 Wheel of Life — Jahresweise Lebensbilanz
- 🪣 Bucket List mit Secret Items
- 📚 Bücherliste mit 3-dimensionaler Bewertung
- 🌙 Traumtagebuch
- ⚖️ Rules of Life: Prinzipien, Affirmationen, No-Gos
- ⬡ Habit Flow & Counter
- 🎲 Würfel-Entscheider

**Alle Daten werden lokal gespeichert. Kein Account, kein Abo, kein Internet erforderlich.**
